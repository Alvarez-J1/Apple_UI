# MacBook Pro Landing Page Clone
A responsive Apple MacBook Pro product landing page clone inspired by Apple’s official website design system.
This project focuses on pixel-perfect UI replication, layout consistency, and responsive behavior across breakpoints.

## Features
1. Pixel-Perfect Apple-Inspired UI
Recreated Apple’s MacBook Pro product page layout, including:

Hero section with product showcase
Navigation bar with Apple-style spacing and typography
Clean dark theme aesthetic
Structured product marketing layout

2. Responsive Layout (Critical Focus)
Fully responsive across large screens, laptops, tablets, and mobile
Maintains layout integrity without breaking below 1200px
Adaptive spacing and alignment similar to Apple’s official site behavior
Optimized for ultra-wide and standard desktop screens

4. Modern UI Structure
Hero product section with layered MacBook visuals
Typography hierarchy matching Apple-style marketing flow
Clean CTA button styling (“Order Now” style interaction)
Feature metric row (CPU, GPU, ML, Battery performance) 

5. Performance & Simplicity
Lightweight static frontend
No framework overhead (pure HTML/CSS/JS approach)
Optimized image usage for fast load times
Minimal DOM complexity for smooth rendering

6. Design Accuracy Focus
Spacing and alignment tuned for visual balance
Consistent grid structure across sections
Footer layout modeled after Apple’s global navigation system

## Tech Stack
